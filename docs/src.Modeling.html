src.Modeling.depletion_computing.compute_distance_and_overlap_length(spatial: dict, logic: dict) → Tuple[float, float]¶

computes distance and overlap lengths between the two wells in question

————— Key of spectral_dict abbreviations —————

—————————– level 0 —————————–

P - Primary well

O - Offset well

—————————– level 1 —————————–

N - North

S - South

M - Slope

O - Orthogonal slope

L - length of well’s lateral in feet

—————————– level 2 —————————–

X - x axis value in feet

Y - y axis value in feet

Long - longitudinal value

Lat - latitudinal value

P - orthogonal projection on other well

—————————– level 3 —————————–

X - x axis value in feed of orthogonal projection

Y - y axis value in feet of orthogonal projection

————————— logic_dict —————————-

‘check_1’: (min(spatial[‘O’][‘S’][‘X’], spatial[‘O’][‘N’][‘X’]) <= spatial[‘P’][‘S’][‘P’][‘X’] <= max(

spatial[‘O’][‘S’][‘X’], spatial[‘O’][‘N’][‘X’])),

‘check_2’: (min(spatial[‘O’][‘S’][‘Y’], spatial[‘O’][‘N’][‘Y’]) <= spatial[‘P’][‘S’][‘P’][‘Y’] <= max(

spatial[‘O’][‘S’][‘Y’], spatial[‘O’][‘N’][‘Y’])),

‘check_3’: (min(spatial[‘O’][‘S’][‘X’], spatial[‘O’][‘N’][‘X’]) <= spatial[‘P’][‘N’][‘P’][‘X’] <= max(

spatial[‘O’][‘S’][‘X’], spatial[‘O’][‘N’][‘X’])),

‘check_4’: (min(spatial[‘O’][‘S’][‘Y’], spatial[‘O’][‘N’][‘Y’]) <= spatial[‘P’][‘N’][‘P’][‘Y’] <= max(

spatial[‘O’][‘S’][‘Y’], spatial[‘O’][‘N’][‘Y’])),

‘check_5’: (min(spatial[‘P’][‘S’][‘X’], spatial[‘P’][‘N’][‘X’]) <= spatial[‘O’][‘S’][‘P’][‘X’] <= max(

spatial[‘P’][‘S’][‘X’], spatial[‘P’][‘N’][‘X’])),

‘check_6’: (min(spatial[‘P’][‘S’][‘Y’], spatial[‘P’][‘N’][‘Y’]) <= spatial[‘O’][‘S’][‘P’][‘Y’] <= max(

spatial[‘P’][‘S’][‘Y’], spatial[‘P’][‘N’][‘Y’])),

‘check_7’: (min(spatial[‘P’][‘S’][‘X’], spatial[‘P’][‘N’][‘X’]) <= spatial[‘O’][‘N’][‘P’][‘X’] <= max(

spatial[‘P’][‘S’][‘X’], spatial[‘P’][‘N’][‘X’])),

‘check_8’: (min(spatial[‘P’][‘S’][‘Y’], spatial[‘P’][‘N’][‘Y’]) <= spatial[‘O’][‘N’][‘P’][‘Y’] <= max(

spatial[‘P’][‘S’][‘Y’], spatial[‘P’][‘N’][‘Y’]))

Parameters

  • spatial (dict) – spatial data of the two well’s points

  • logic (dict) – checks between two well’s which drives how the avg_2d_distance and overlap_length are computed

Returns

the average 2-d distance between the two well’s laterals, and the perpendicular-to-lateral overlap length
