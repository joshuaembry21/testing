# Data sheets: Info - description of producing group

IPPF Quartile 1 (OQ1 and/or GQ1) IPPF Quartile 2 (OQ2 and/or GQ2) IPPF Quartile 3 (OQ3 and/or GQ3) IPPF Quartile 4 (OQ4 and/or GQ4) PF - projected PF summary table SDF - projected SDF summary table

# Columns (Q1 to Q4): Month, 2010-2020 PFs, Projected PF,

Month, 2010-2020 STDEV, Projected STDEV, Month, 2010-2020 WCs

# Calculations: Projected PF line (using Arp’s Equation)

Projected SD line (using linear interpolation) Average 12-month well count

# Chart sheets: Production Factors - Bar plot (first 36 months)

Production Factors - Scatter or XY plot (first 84 months) Production Factors - Bar plot (first 120 months) Standard Deviation Factors - Bar plot (first 36 months) Standard Deviation Factors - Scatter or XY plot (first 84 months) Wellcounts - Scatter or XY plot (average 12-month well count) Oil Production Factors Projection - Scatter plot (first 84 months) Gas Production Factors Projection - Scatter plot (first 84 months)
