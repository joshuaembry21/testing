cli-production_factor_analysis module¶

Production Factor Analysis (Main)

Input data:

data_path - location of data folder within the project directory; type: ‘str’ prod_zone: producing zone(s); type: ‘str’ or [‘str’] –group: production factor group index; type ‘str’; default: ‘0’ –single-product: whether to treat the zone as a single product zone; type: bool; default: False –first-prod-year: lower bound to use for data pull; type: int; default: 2010 –last-prod-year: upper bound to use for data pull; type: int; default: 2020 –dont-discard-nan-prod: whether to ignore nan production values; type: bool; default: False –dont-save-plots-to-pdf: whether to save production factor plots to pdf; type: bool; default: False –dont-save-results-to-excel: whether to save production factor results to excel; type: bool; default: False –raw-data-to-excel: whether to output raw data to excel; type: bool; default: False –dont-overwrite-projection: whether to overwrite both PF and SDF projections; type: bool; default: False

Some valid producing zone names by basin/play:

anadarko - Woodford, Meramec Upper, Meramec Lower, Osage Lower appalachian - Utica_Point Pleasant, Marcellus arkoma - Fayetteville, Woodford delaware - Wolfcamp A Lower, Wolfcamp A XY, Wolfcamp B, Bone Spring Second Sand, Bone Spring Third Sand denver_julesburg - Niobrara A, Niobrara B, Niobrara C east_texas_arkla - Haynesville Upper, Haynesville Lower, Bossier Lower, Cotton Valley Sands fort_worth - Barnett Shale gulf_coast - Eagle Ford Lower, Eagle Ford Upper, Austin Chalk midland - Wolfcamp A, Wolfcamp B, Lower Spraberry powder_river - Turner-Frontier, Parkman williston - Bakken, Three Forks
